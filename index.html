Projects

PowerApps Project 1: Employee Directory

This PowerApps project is an Employee Directory app that allows users to search for employees by name, department, or role. It integrates with Microsoft Azure Active Directory to fetch employee details, including contact information and organizational hierarchy. The app provides an easy-to-use interface for managing and updating employee data.

PowerApps Project 2: Expense Tracker

The Expense Tracker app helps employees submit and manage their expense reports. Users can upload receipts, categorize expenses, and track the approval status of their submissions. The app integrates with Microsoft SharePoint for data storage and uses Power Automate for workflow automation, ensuring a smooth approval process.

Power Automate Project 1: Automated Email Notifications

This Power Automate project is designed to send automated email notifications to employees based on certain triggers. For example, when a new document is uploaded to a SharePoint library, the workflow sends an email to relevant stakeholders. The flow also includes error handling and logging to ensure reliable operation.

Power Automate Project 2: Data Synchronization

The Data Synchronization flow connects multiple data sources, such as SQL Server and SharePoint, to keep information consistent across platforms. This automation ensures that updates made in one system are reflected in the other, reducing data redundancy and improving accuracy. The project includes scheduled runs and error notifications to maintain data integrity.
